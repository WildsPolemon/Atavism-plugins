# StarNet Core — повний аналог AinurPOS

Джерела: [ainurpos.com.ua](https://ainurpos.com.ua/), `AinurPOS_Список_функцій_Частина_1.docx`

## Модулі (як у AinurPOS)

### POS — Точка продажу
- Продаж в магазині (Web / PC)
- Товари, послуги, комплекти на одному екрані
- Пошук товарів + сканер штрихкодів
- Пошук клієнтів (ім'я, телефон, картка)
- Знижки (на чек / на позицію)
- Відкладений чек (park sale)
- Комбінована оплата (готівка + картка)
- Повернення / відшкодування
- Примітки до чека
- X/Z-звіти, відкриття/закриття зміни
- Друк / email чеків (шаблони)

### Склад / Inventory
- Надходження, закупівлі, постачальники
- Переміщення між складами
- Списання, коригування
- Інвентаризація
- Багатоскладський облік
- Контроль залишків, собівартість
- Штрихкоди, цінники, етикетки

### Товари
- Категорії / підкатегорії, варіанти, SKU
- Вагові товари, PLU, фото, опис
- Ціни: закупівельна, роздрібна, акційна
- Масове редагування, імпорт/експорт Excel
- **Автопошук за штрихкодом** (локальна БД + Open Food Facts)

### CRM
- База клієнтів, історія покупок
- Програми лояльності, накопичувальні знижки
- Контроль боргів, імпорт/експорт

### Звіти
- Продажі, прибуток, склад, працівники, фінанси
- Конструктор звітів (базовий)

### eStore (майбутнє)
- Онлайн-вітрина, інтеграція з POS

### Тарифи (референс AinurPOS)
- Базовий $0, Ощадливий $9, Оптимальний $19, Професійний $49, Необмежений $89
