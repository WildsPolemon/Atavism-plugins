# Дослідження AinurPOS

## Важливо: два різні продукти

| URL | Що це | Навіщо |
|-----|--------|--------|
| **ainurpos.com.ua** | Маркетинговий сайт | Список функцій, тарифи, опис модулів |
| **web.ainur.app/pos** | **Веб-POS (каса)** | **Еталон UI для StarNet Core каси** |
| **web.ainur.app** | Адмін-портал (склад, звіти) | Еталон для адмін-панелі |

StarNet Core **не копіює** маркетинговий сайт. Каса (`:5175`) зроблена за макетом **веб-POS** AinurPOS:
- світла тема
- зліва: пошук + категорії + сітка товарів
- справа: клієнт + таблиця чека + зелена кнопка РАЗОМ
- логін: «Точка продаж» / «Авторизація»

## Детальний список функцій AinurPOS (з сайту)

### POS — [pos-dodatok](https://ainurpos.com.ua/pos-dodatok/)
- Товари, послуги, набори на одному екрані
- Пошук товарів + сканер штрихкодів
- Пошук клієнтів (ім'я, телефон, дисконтна картка)
- Примітки на чеку
- Знижки (на весь чек / на позицію)
- Індивідуальні чеки (логотип, шаблони, email, друк)
- Відкладений продаж (park sale)
- Розділений платіж (готівка + картка)
- Повернення / відшкодування
- Web + Android + iOS + Windows
- POS-обладнання: принтери, сканери, SUNMI

### Склад — [programma-dlya-torgovly-y-sklada](https://ainurpos.com.ua/programma-dlya-torgovly-y-sklada/)
- Облік залишків
- Управління закупівлями (email постачальнику в 1 клік)
- Списання товару
- Інвентаризація (мобільно)
- Переміщення між складами
- Кілька складів / міст
- Складські звіти
- Автоматичний розрахунок собівартості
- Оцінка складу
- Друк штрихкодів і цінників
- Мобільний додаток для складу

### CRM — [customers-management](https://ainurpos.com.ua/customers-management/)
- База клієнтів в одному місці
- Імпорт і експорт
- Контроль боргів
- Історія покупок
- Накопичувальні знижки (лояльність)
- Рекомендації товарів
- Мобільний CRM

### eStore (головна сторінка)
- Онлайн-магазин за кілька кліків
- Інтеграція з POS і складом
- Самовивіз / доставка
- Банери, аналітика

### Звіти (головна + тарифи)
- Продажі в реальному часі
- Прибуток, витрати
- Конструктор звітів (Standard report builder — з тарифів)
- Звіти по працівниках

### Тарифи
- Базовий $0 · Ощадливий $9 · Оптимальний $19 · Професійний $49 · Необмежений $89
