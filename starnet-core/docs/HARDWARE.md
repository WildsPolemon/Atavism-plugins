# Обладнання каси StarNet Core

**Каса → Меню → Налаштування → Обладнання**

Просте підключення як у AinurPOS — без мостів і додаткових програм.

## Терези (USB)

1. Підключіть ваги до комп'ютера через USB
2. Увімкніть **Ваги** в налаштуваннях
3. Натисніть **Знайти і підключити ваги**
4. Браузер (Chrome/Edge) покаже доступний пристрій — оберіть його один раз

Після першого підключення ваги запам'ятовуються. При додаванні **вагового товару** вага зчитується автоматично.

## Принтер чека (WiFi)

1. Підключіть чековий принтер до тієї ж WiFi-мережі, що й каса
2. Увімкніть **Друк чеків**
3. Натисніть **Знайти принтер у WiFi** — система просканує мережу
4. Оберіть знайдений принтер або вкажіть IP вручну
5. **Тест друку** для перевірки

Альтернатива: підключення через **USB** (кнопка «Підключити USB-принтер»).

## Термінал Privat24

1. Увімкніть **Оплата карткою**
2. Підключіть термінал до WiFi (IP — необов'язково)
3. На екрані оплати натисніть **Оплата через термінал Privat24**
4. Проведіть оплату на терміналі та підтвердіть у касі

## Сканер штрих-кодів

Працює як USB-клавіатура — просто скануйте товар, код з'явиться в полі пошуку.

## Вимоги

- Браузер: **Chrome** або **Edge** (для ваг і USB-принтера)
- Каса та принтер/термінал в одній **WiFi-мережі**
