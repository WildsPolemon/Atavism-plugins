# Архіви релізу StarNet Core для хостингу

| Файл | Опис |
|------|------|
| `starnet-core-hosting.zip` | Повний пакет: API + адмін + каса + інсталятор + демо |

## Встановлення

1. Завантажте ZIP на хостинг
2. Розпакуйте в `public_html/`
3. Відкрийте `https://ваш-домен/install/`
4. Пройдіть 2 кроки — готово

Перезбірка: `scripts/build-release.sh`
